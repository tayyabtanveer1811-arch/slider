Flutter UI Assignment

Description

This project is a simple Flutter user interface created for a mobile application development assignment. It demonstrates the use of basic Flutter widgets such as Container, Column, Image, and ListView to design a clean and scrollable layout.
The interface includes a header container with the student's name and registration number, a main image, a horizontal row of multiple images, and a footer container. The layout uses vertical and horizontal scrolling to improve user experience.
Features:
Header container with name and registration number
Main image display
Horizontal scrollable image gallery
Footer container
Simple and clean Flutter UI layout
Technologies Used:
Flutter
Dart
